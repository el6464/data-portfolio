# Projet 2 : Automatisation du diagnostic permanent

## Objectif
Créer une solution Power BI pour automatiser le diagnostic permanent (DIAG PERM 21) et obtenir des indicateurs fiables.

## Technologies utilisées
- Power BI
- Power Query
- PostgreSQL
- QGIS

## Étapes et méthodologie
1. Importer les données de l’EDD et MySIG
2. Créer des mesures DAX pour calcul des indicateurs Diagperm
3. Répartition correcte des données géolocalisées
4. Export standardisé des tableaux pour DIAG PERM 21
