# Projet 4 : Suivi des lignes dans l’EDD

## Objectif
Surveiller le volume et l’évolution des données dans l’Entrepôt de Données pour garantir leur fiabilité et pertinence.

## Technologies utilisées
- PostgreSQL
- Power BI
- Power Query

## Étapes et méthodologie
1. Extraction des données et dates de mise à jour via requêtes SQL
2. Intégration dans Power BI pour visualisation dynamique
3. Suivi régulier des anomalies et doublons
